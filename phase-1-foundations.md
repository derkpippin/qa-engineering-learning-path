# Phase 1: Foundations & Git/GitHub Basics

## Learning Objectives
- Understand the fundamental concepts of version control.
- Learn the basics of Git, including installation, configuration, and commands.
- Familiarize yourself with GitHub as a hosting platform for Git repositories.
- Collaborate on projects using Git and GitHub.

## Tutorials
1. **Introduction to Version Control**
   - Explore the importance of version control systems in software development.
2. **Getting Started with Git**
   - [Git Installation and Setup](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
3. **Basic Git Commands**
   - Learn commands such as `git init`, `git clone`, `git add`, `git commit`, and `git push`.
4. **Understanding Branches**
   - A tutorial on branching and merging in Git.
5. **GitHub Basics**
   - [GitHub Guide](https://guides.github.com/introduction/git-handbook/) for beginners.

## Hands-On Exercises
- **Exercise 1**: Initialize a new Git repository and make your first commit.
- **Exercise 2**: Create a branch, make changes, and merge back to main.
- **Exercise 3**: Collaborate on a shared repository on GitHub.
